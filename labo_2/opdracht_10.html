Opdracht 10

In de oorspronkelijke code was de nesting van de lijsten incorrect. Een geneste lijst moet altijd binnen een <li> element geplaatst worden.

De incorrecte versie was:
<ul>
  <li>een</li>
  <li>twee</li>
  <ol>
    <li>jan</li>
    <li>piet</li>
    <li>joris</li>
  </ol>
  <li>vier</li>
</ul>

De correcte versie is:
<ul>
  <li>een</li>
  <li>twee
    <ol>
      <li>jan</li>
      <li>piet</li>
      <li>joris</li>
    </ol>
  </li>
  <li>vier</li>
</ul>
