Opdracht 3

De generator geeft de <tr> elementen van elke even rij een 'alt' class. De generator voorziet de cellen in deze rijen van een andere achtergrondkleur, te zien in de regel:

.datagrid table tbody .alt td { background: #E1EEF4; color: #00496B; }

Daarnaast heb ik enkele cellen voorzien van een 'onbekend' class, met de volgende CSS regel:

.onbekend { background-color: red; }

Het effect van de 'onbekend' class werkt niet op cellen binnen een <tr> met de 'alt' class. Dit komt doordat de specificiteit van de '.datagrid table tbody .alt td' rule hoger is dan de '.onbekend' rule. De 'alt' class heeft meer selectorkracht en overschrijft daarom de rode achtergrond.
