Opdracht 3

Waarom staat de tekst horizontaal gecentreerd?

De tekst staat horizontaal gecentreerd door de CSS-regel text-align: center op het div-element.

Vertical-align: center werking

  • vertical-align: center heeft geen effect omdat deze property niet bestaat
  • vertical-align: middle werkt niet voor het centreren van tekst binnen een element
  • vertical-align bepaalt hoe een element zich uitlijnt ten opzichte van andere elementen in dezelfde regel, niet de inhoud binnen het element zelf

Simpele manieren voor verticale centrering:

Voor 1 regel tekst:

Zet de line-height gelijk aan de height van het element:

.area {
  height: 100px;
  line-height: 100px;
}

Voor meerdere regels tekst:

Gebruik display: table-cell met vertical-align: middle:

.area {
  display: table-cell;
  vertical-align: middle;
}

CSS Specificity vraag

De class tall krijgt voorrang omdat:

  • Beide regels hebben dezelfde specificiteit (class selector)
  • De regel voor class tall komt later in het CSS-bestand
  • Bij gelijke specificiteit wint de laatst gedefinieerde regel
