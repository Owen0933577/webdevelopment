Waarom zijn er lege ruimtes tussen de blokken?

  • HTML witruimte tussen inline-block elementen wordt ge√Ønterpreteerd als een spatie in de weergave
  • Dit gebeurt omdat inline-block elementen zich gedragen als woorden in een tekst
  • HTML behandelt witruimte tussen deze elementen net zoals witruimte tussen woorden

Oplossingen voor het verwijderen van de witruimte:

1. HTML elementen direct na elkaar plaatsen:

<div class="area">01</div><div class="area">02</div>

2. Negatieve margin gebruiken:

.area { margin-right: -4px; }

3. Font-size: 0 op de parent zetten:

parent { font-size: 0; } .area { font-size: 16px; }

Responsive Gedrag

De blokken worden automatisch verdeeld over meerdere regels omdat:

  • Inline-block elementen zich gedragen als woorden in een tekst
  • Als er niet genoeg ruimte is op een regel, springt het volgende element automatisch naar de volgende regel
  • Dit gedrag is ingebouwd in de browser en heeft geen extra CSS voor basic responsive layout
