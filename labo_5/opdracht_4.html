Typische verschillen in broncode van vergelijkbare designs:

  • Meta tags:
    • Verschillende meta descriptions
    • Andere keywords
    • Specifieke social media tags
  • CSS referenties:
    • Andere bestandsnamen voor stylesheets
    • Verschillende volgorde van CSS imports
    • Inline styles vs. externe stylesheets
  • JavaScript:
    • Verschillende script bronnen
    • Andere volgorde van script tags
    • Inline scripts vs. externe bestanden
  • Class namen:
    • Verschillende naamgevingsconventies
    • Framework-specifieke classes (Bootstrap vs. Tailwind)
  • HTML structuur:
    • Andere div nesting
    • Verschillende gebruik van semantische tags
    • Alternatieve container structuren

Hoe te vergelijken:

  1. Open twee verschillende designs in aparte browser tabs
  2. Rechtsklik op elke pagina en selecteer 'View page source'
  3. Plaats de vensters naast elkaar
  4. Let specifiek op:
    • De HEAD sectie voor verschillende meta-informatie
    • De structuur van de BODY sectie
    • Gebruikte frameworks en bibliotheken
    • Commentaar en documentatie in de code
